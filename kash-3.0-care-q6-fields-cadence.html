OPTION 1

No cadence — title + theme only

Practices are just named acts. You check them off whenever. Simplest possible model.

Drop the Cadence field entirely.

OPTION 2

Soft cadence label

An optional rhythm shown as a gentle hint under the practice ("most days"). Purely descriptive — never nags, never scores.

Stores a label; powers display only. Sets up "what lifts me" later (option B) without committing now.

OPTION 3

Cadence that pre-fills scheduling

The same field, but when you "Add to my day," the cadence pre-fills the repeat rule (a "daily" practice → a daily repeat) via your recurrence engine.

Most reuse of the spine; a touch more wiring. Still just a default you can override.
