Tokens · A1 (round 2) — Reuse the existing palette for projects

All four schemes reuse only colors already in the system — no new hues. They differ in whether category meaning survives in project-mode.

Scenario: Client site (Professional) · Internal tool (Professional — same category!) · Book (Personal Projects) · Marathon (Body & Mind). Watch how each scheme handles the two Professional projects.
