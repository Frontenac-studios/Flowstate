Thread 2 · Do protected blocks count toward over-commit load?
A protected block is spoken-for time. Should it add to a day’s load (so over-protecting can trip the warning), or sit outside it? Wellbeing wrinkle: most protected blocks are restorative (Relationships, Body & Mind) — counting them could discourage protecting them.
2A · Count fully

Protected time fills the day

tasks + protected = load nears threshold
Total commitment = tasks + protected. Protect too much → warned.
  • True “how full is my day”
  • Penalizes protecting wellbeing time
2B · Reduced weight

Counts, but lighter than work

protected = ½ weight room to spare
Protected time consumes some capacity (it’s real) but at reduced weight, so it rarely trips the warning alone.
  • Honest about capacity
  • Doesn’t punish self-care
  • A weight factor to pick
2C · Excluded

protected ignored well clear

  • Simplest; encourages protecting
